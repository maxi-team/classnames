# @mntm/classnames [![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/maxi-team/classnames/blob/master/LICENSE) [![npm bundle size](https://img.shields.io/bundlephobia/min/@mntm/classnames)](https://bundlephobia.com/result?p=@mntm/classnames)

A simple javascript utility for conditionally joining classNames together.

## Installation

We recommend to use [yarn](https://classic.yarnpkg.com/en/docs/install/) for dependency management:

```shell
yarn add @mntm/classnames
```
